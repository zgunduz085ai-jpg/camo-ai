# CAMO.AI Dashboard

AI destekli CAMO yönetim platformu.

## Vercel'e Yükleme (Ücretsiz)

### Yöntem 1 — GitHub üzerinden (Önerilen)

1. [github.com](https://github.com) → New repository → "camo-ai" adını verin
2. Bu klasördeki tüm dosyaları yükleyin
3. [vercel.com](https://vercel.com) → Sign up (GitHub ile)
4. "New Project" → GitHub reponuzu seçin
5. Deploy → birkaç dakikada siteniz hazır!

### Yöntem 2 — Vercel CLI

```bash
npm install -g vercel
cd camo-app
vercel
```

## Lokal Çalıştırma

```bash
npm install
npm start
```

## Özellikler
- ✈ Filo yönetimi (uçak ekleme, düzenleme)
- ⚙️ AMP Analizi AI asistanı
- 📋 AD Takibi
- 📖 AMM/SRM Doküman Botu
- ⚠️ Arıza Analizi
- 🔩 LLP Takibi
- 📂 Google Drive klasör entegrasyonu
- 📊 Excel/CSV dosya yükleme
